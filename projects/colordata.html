ColorData — Mobile App for Color Identification

2010-12-23

"Color Data is an app designed to assist you with remembering colors that you like, as well as identifying color values that are hard to distinguish. It has an intuitive interface. It lets you snap a photo or grab one from your library. Once you have an image, you can zoom in and find the exact color you want. Color Data not only provides a way to save the color, but also provides information about color numerics and even harmonies."

— App description from the Windows Phone Marketplace, December 2010

Concept & Motivation

Though I am colorblind, I find myself drawn to issues of color. I believe that it is because of the straightforward way a computer can quantify color. It lets me be a little closer to understanding them myself and in the process, hopefully, help others with color-related issues.

ColorData transforms the subjective experience of color into objective data. For someone who experiences color differently, the precision of RGB values, hex codes, HSL measurements, and CMYK percentages provides a reliable framework for working with and communicating about color.

Key Features

Photo Capture & Selection

Users can snap a new photo or select from their existing library. The zoom functionality enables precise pixel-level color selection, ensuring accurate color capture even in complex images.

Color Information Display

The app presents color data in multiple formats:

  • RGB Info: Red, Green, Blue values (0-255 range)
  • HLV Info: Hue (0-360°), Lightness, Value percentages
  • CMYK Info: Cyan, Magenta, Yellow, Key (black) percentages
  • Hexadecimal: Standard web color format (#RRGGBB)

Split Complement Color Harmonies

Beyond individual color identification, ColorData calculates complementary and split-complementary color harmonies. This feature assists users in developing cohesive color palettes based on color theory principles.

Color Sets & Collections

Users can organize colors into named sets with descriptions and notes. This organizational system enables saving color palettes for specific projects, tracking colors seen in the world, or building reference libraries for design work. Examples from user reviews include sets for:

  • Monarch butterfly wing colors
  • Bluejay plumage
  • Lawn chair fabrics
  • John's car paint
  • Website design palettes

Tweek Screen

The "tweek" (fine-tune) screen allows users to adjust color selections after capture. This accounts for device camera variations and lighting conditions, ensuring the saved color matches the user's intent rather than just the captured pixel.

User Reception

ColorData received positive reviews on the Windows Phone Marketplace, with users appreciating the intuitive interface and regular updates. Key feedback themes:

Technical Implementation

Developed for Windows Phone using C# and the Windows Phone SDK. The app handles:

  • Camera integration and photo library access
  • Image manipulation and zoom controls
  • Color space conversions (RGB ↔ HSL ↔ CMYK ↔ Hex)
  • Color harmony calculations
  • Local data storage for color sets
  • Touch-based UI optimized for mobile interaction

Reflection

ColorData represents an intersection of personal need and technical capability. By approaching color as data rather than purely visual experience, the app makes color accessible to users who perceive it differently. The project demonstrates how computational thinking can bridge perceptual differences and create practical tools for everyday challenges.

The positive user reception and ongoing feature requests validated the concept: quantifying color serves both accessibility needs and practical use cases like design work, shopping, and color matching.
